# Walmart Sales Forcasting

## Starting Guide
### 1. Clone the repo
```bash
git clone <repo-url>
cd <repo-folder>
```
---
### 2. Create Venv and Activate
#### Linux and MacOS
```bash
python3 -m venv .venv
source .venv/bin/activate
```
#### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
```
---
### 3. Install Libraries
```bash
pip install -r requirements.txt
```
